## MSP_EXP432E40Y FreeRTOS Examples
This repository contains additional example(s) for programming Simplelink MCUs using FreeRTOS. 

All the projects have Dependencies on Referenced Project - freertos_builds_MSP_EXP432E401Y_release_css. The projects are modified after cloning the empty project provided in the Simplelink SDK for FreeRTOS system.

Please change the location of the FREERTOS_INSTALL_DIR from the properties > Resource > Linked Resources of Code Composer Studio to your local install directory of FreeRTOS.

# Requirements - 
1. Code Composer Studio (Version 9.0.1.00004 used)
2. FreeRTOS (Version 10.2.0 used)
3. MSP_EXP432E40Y Rev - 1.0

# Important
*The code has not yet been tested in a different machine - cloned from git repo. If not able to compile, please clone the empty FreeRTOS project from local projects installed in Simplelink SDK examples and copy the source files from the git repo*
